# Examples - Thai Decision Memo

## Example input
กำลังตัดสินใจว่าจะเริ่มโครงการ AI chatbot กับลูกค้าด้วย rollout แบบเต็มหรือเริ่ม pilot ก่อน ทีมขายอยากเริ่มเร็ว แต่ทีม operation กังวลเรื่อง readiness

## Example output
### Decision Needed
ตัดสินใจเลือกรูปแบบการเริ่มโครงการระหว่าง rollout แบบเต็มกับการเริ่ม pilot ก่อน

### Recommendation
แนะนำให้เริ่มในรูปแบบ pilot ก่อน เพื่อควบคุมความเสี่ยงและให้ทีม operation มีเวลาปรับ process รองรับ
