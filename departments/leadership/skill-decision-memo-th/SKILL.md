# Thai Decision Memo

Turn rough decision context into a clear Thai memo.

## Output requirements
- Write in Thai
- Keep the memo concise and decision-oriented
- Present recommendation clearly
- Show trade-offs honestly
- Avoid hiding uncertainty

## Working method
1. Clarify the decision to be made
2. Extract options and trade-offs
3. Identify recommendation and reasoning
4. Highlight risks or dependencies
5. End with the required decision or next step

## Default output structure
1. decision needed
2. recommendation
3. rationale
4. alternatives considered
5. risks / trade-offs
6. next step
