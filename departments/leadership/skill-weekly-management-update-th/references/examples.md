# Examples - Thai Weekly Management Update

## Example input
สัปดาห์นี้ทีมขายปิดดีลได้ 2 ราย ทีมการตลาดปล่อยคอนเทนต์ชุดใหม่แล้ว แต่ทีม operation ยังติดเรื่อง process onboarding ลูกค้าใหม่ที่ยังไม่ลงตัว

## Example output
### Weekly Summary
สัปดาห์นี้ภาพรวมมีความคืบหน้าในเชิงยอดขายและการตลาด แต่ยังมีประเด็นด้าน operation ที่ควรเร่งจัดระบบเพื่อรองรับการเติบโตต่อเนื่อง

### Priorities Next Week
- ปรับ process onboarding ลูกค้าใหม่
- ติดตาม conversion จากคอนเทนต์ชุดล่าสุด
