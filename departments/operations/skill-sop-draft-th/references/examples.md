# Examples - Thai SOP Draft

## Example input
ขั้นตอนรับ lead จากฟอร์มเว็บไซต์: มี notification เข้า LINE กลาง, แอดมินเช็กข้อมูลเบื้องต้น, ส่งต่อให้เซลส์, ถ้าข้อมูลไม่ครบให้ติดต่อกลับภายในวันเดียวกัน, ถ้าครบให้นัดคุย

## Example output
### วัตถุประสงค์
เพื่อให้การรับ lead จากฟอร์มเว็บไซต์เป็นไปอย่างรวดเร็วและไม่ตกหล่น

### ขั้นตอนการทำงาน
1. เมื่อมี lead ใหม่จากฟอร์มเว็บไซต์ ระบบจะแจ้งเตือนไปยัง LINE กลาง
2. แอดมินตรวจสอบข้อมูลเบื้องต้นของ lead
3. หากข้อมูลไม่ครบ ให้ติดต่อกลับภายในวันเดียวกันเพื่อขอข้อมูลเพิ่มเติม
4. หากข้อมูลครบ ให้ส่งต่อให้ฝ่ายขายและดำเนินการนัดคุย
