# Examples - Thai Internal Announcement

## Example input
ต้องการประกาศภายในให้ทีมทราบว่า วันจันทร์หน้าจะมีการย้ายระบบแชทลูกค้าไปยังระบบใหม่ และอาจมีช่วงที่ใช้งานไม่ได้ประมาณ 1 ชั่วโมง

## Example output
### ประกาศภายใน: แจ้งย้ายระบบแชทลูกค้า
เรียนทีมงานทุกท่าน

ขอแจ้งให้ทราบว่าในวันจันทร์หน้า ทางบริษัทจะดำเนินการย้ายระบบแชทลูกค้าไปยังระบบใหม่ โดยในช่วงดำเนินการอาจมีผลให้ระบบใช้งานไม่ได้ประมาณ 1 ชั่วโมง

หากมีข้อมูลเพิ่มเติม ทีมงานที่เกี่ยวข้องจะอัปเดตให้ทราบอีกครั้ง
