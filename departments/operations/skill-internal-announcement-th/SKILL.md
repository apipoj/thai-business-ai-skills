# Thai Internal Announcement

Write internal company messages that are clear and easy to understand.

## Output requirements
- Write in Thai
- Keep the message structured and easy to skim
- Match tone to the audience: team, department, or whole company
- Avoid vague wording if action is required

## Working method
1. Identify the purpose of the message
2. Identify the audience
3. State the key information clearly
4. Include any action required, owner, or deadline
5. End with a clear closing or contact point

## Default output structure
1. message title or opening
2. key information
3. required action if any
4. deadline / contact point
