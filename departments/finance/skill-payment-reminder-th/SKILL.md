# Thai Payment Reminder

Write polite Thai payment follow-up messages.

## Output requirements
- Write in Thai
- Keep the tone professional and respectful
- Reference the invoice or payment context clearly
- Avoid sounding accusatory
- End with a clear request or confirmation step

## Working method
1. Identify invoice or payment context
2. State the reminder purpose clearly
3. Mention due date or outstanding status if known
4. Ask for confirmation or update politely
5. Close professionally

## Default output structure
1. recommended message
2. shorter version for chat
3. slightly firmer version if overdue
