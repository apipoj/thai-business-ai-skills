# Examples - Thai Payment Reminder

## Example input
ต้องการติดตาม invoice เลขที่ INV-024 ที่ครบกำหนดชำระเมื่อ 3 วันที่แล้ว แต่ยังไม่ได้รับการโอน

## Example output
สวัสดีครับ ขออนุญาตติดตามการชำระเงินสำหรับ invoice เลขที่ INV-024 ซึ่งครบกำหนดชำระแล้วเมื่อ 3 วันที่ผ่านมานะครับ ไม่ทราบว่าทางทีมสะดวกอัปเดตสถานะการชำระให้ได้หรือไม่ครับ

หากมีข้อมูลเพิ่มเติมหรือต้องการเอกสารประกอบเพิ่มเติม แจ้งได้เลยครับ
