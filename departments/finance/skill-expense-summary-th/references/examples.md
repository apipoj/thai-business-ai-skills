# Examples - Thai Expense Summary

## Example input
ค่าใช้จ่ายเดือนนี้มีค่าโฆษณา 45,000 บาท ค่าเครื่องมือ SaaS 18,000 บาท ค่าจ้างฟรีแลนซ์ 22,000 บาท และค่าเดินทาง 9,000 บาท โดยค่าโฆษณาเพิ่มจากเดือนก่อนค่อนข้างมาก

## Example output
### สรุปภาพรวม
ค่าใช้จ่ายหลักของเดือนนี้กระจุกอยู่ที่ค่าโฆษณา ค่าจ้างฟรีแลนซ์ และค่าเครื่องมือ SaaS โดยค่าโฆษณาเป็นหมวดที่สูงที่สุดและเพิ่มขึ้นจากเดือนก่อนอย่างมีนัยสำคัญ

### Items to Review
- ตรวจสอบสาเหตุที่ค่าโฆษณาเพิ่มขึ้น
- ประเมินความคุ้มค่าของเครื่องมือ SaaS ที่ใช้งานอยู่
