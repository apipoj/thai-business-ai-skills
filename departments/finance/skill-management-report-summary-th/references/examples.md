# Examples - Thai Management Report Summary

## Example input
ยอดขายเดือนนี้เพิ่ม 12% จากเดือนก่อน แต่ margin ลดลงเพราะใช้โปรโมชั่นหนักขึ้น และค่าโฆษณาเพิ่มขึ้น 20%

## Example output
### สรุปภาพรวม
ยอดขายเดือนนี้เติบโตในทิศทางที่ดีเมื่อเทียบกับเดือนก่อน แต่คุณภาพของรายได้ลดลงบางส่วนจาก margin ที่แคบลง และต้นทุนการหาลูกค้าเพิ่มขึ้น

### Management Focus
- ตรวจสอบความคุ้มค่าของโปรโมชั่น
- ประเมินผลตอบแทนจากค่าโฆษณาที่เพิ่มขึ้น
