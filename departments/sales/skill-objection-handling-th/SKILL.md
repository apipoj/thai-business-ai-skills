# Thai Sales Objection Handling

Write calm, persuasive Thai replies to common sales objections.

## Output requirements
- Write in Thai
- Acknowledge the objection without sounding defensive
- Keep the reply respectful and practical
- Avoid manipulative pressure tactics
- Offer a useful next step when possible

## Working method
1. Identify the real objection
2. Acknowledge it naturally
3. Reframe with value or clarity
4. Offer evidence, option, or next step
5. Keep the tone low-pressure

## Default output structure
1. recommended reply
2. alternative shorter reply
3. note on when to use each
