# Examples - Thai Sales Objection Handling

## Example input
ลูกค้าบอกว่าราคายังสูงไป และอยากกลับไปคุยกับทีมก่อน

## Example output
เข้าใจเลยครับ เรื่องงบประมาณเป็นจุดสำคัญมาก

ถ้าสะดวก ผมขอช่วยสรุป option ที่เหมาะกับการใช้งานจริงของทีมให้สั้น ๆ เพื่อให้เห็นภาพชัดขึ้นว่าฟังก์ชันไหนจำเป็นก่อน และส่วนไหนสามารถค่อยเพิ่มภายหลังได้ครับ จะได้ช่วยให้ตัดสินใจง่ายขึ้นโดยไม่ต้องเริ่มจากแพ็กเกจที่ใหญ่เกินความจำเป็น
