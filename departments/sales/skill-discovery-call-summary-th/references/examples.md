# Examples - Thai Discovery Call Summary

## Example input
ลูกค้าใช้ทีมแอดมินตอบแชทเอง ต้องการลดเวลาตอบลูกค้าและอยากเช็กว่าระบบ AI จะช่วยได้แค่ไหน กังวลเรื่องความแม่นยำและการเชื่อมต่อกับ LINE OA

- ตอบลูกค้าช้าในบางช่วงเวลา

### Constraints / Objections

- ต้องการรู้ว่าระบบเชื่อมกับ LINE OA ได้หรือไม่
