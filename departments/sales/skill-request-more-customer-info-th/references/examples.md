# Examples - Thai Request More Customer Information

## Example input
- ลูกค้า: บริษัทขายอุปกรณ์การแพทย์
- สิ่งที่ยังขาด: จำนวนผู้ใช้งาน, ช่องทางที่ต้องการเชื่อมต่อ, use case หลัก, งบประมาณคร่าว ๆ
- เหตุผลที่ต้องถาม: เพื่อประเมิน solution ที่เหมาะสมและจัดทำข้อเสนอราคา
- ช่องทาง: LINE
- โทน: สุภาพ มืออาชีพ แต่ไม่ทางการเกินไป

## Example output
สวัสดีครับ ขอบคุณมากครับสำหรับข้อมูลเบื้องต้น

เพื่อให้ผมช่วยประเมินแนวทางที่เหมาะสมและจัดทำข้อเสนอได้ตรงกับการใช้งานจริงมากขึ้น รบกวนขอข้อมูลเพิ่มเติมเล็กน้อยครับ

1. จำนวนผู้ใช้งานโดยประมาณ
2. ช่องทางที่ต้องการเชื่อมต่อ เช่น LINE, เว็บไซต์ หรือ Facebook
3. ลักษณะการใช้งานหลักที่อยากให้ระบบช่วย
4. งบประมาณคร่าว ๆ ที่วางไว้ (ถ้าสะดวก)

ถ้าสะดวกตอบเป็นข้อสั้น ๆ ได้เลยครับ แล้วผมจะช่วยสรุปแนวทางต่อให้ทันทีครับ

## Reusable prompt pattern
ช่วยเขียนข้อความขอข้อมูลเพิ่มเติมจากลูกค้าเป็นภาษาไทย

ข้อมูลมีดังนี้:
- ลูกค้าคือ: ...
- ข้อมูลที่ยังขาด: ...
- เหตุผลที่ต้องขอข้อมูลนี้: ...
- ช่องทางที่ต้องการใช้: ...
- โทนที่ต้องการ: ...

เงื่อนไข:
- เขียนให้สุภาพ เข้าใจง่าย
- อย่าทำให้ลูกค้ารู้สึกว่าถูกโยนงานกลับ
- ถ้ามีหลายข้อ ให้จัดเป็น bullet หรือ numbered list
- ปิดท้ายด้วย next step ที่ชัดเจน
- ช่วยเขียน 3 เวอร์ชัน
