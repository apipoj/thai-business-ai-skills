# Thai Request More Customer Information

Write clear, polite Thai messages that ask customers for missing information so the team can move the opportunity forward.

## Output requirements

- Write in Thai
- Keep the message clear and easy to answer
- Be polite and professional without sounding stiff
- Make the request feel helpful, not like extra work being pushed to the customer
- If multiple items are needed, organize them so the customer can reply easily

## Working method

1. Identify what information is missing
2. Identify why that information is needed
3. Group the requested items in a simple, easy-to-answer format
4. Write the request in a tone that feels helpful and low-friction
5. End with a clear next step

## Tone guidance

Prefer Thai business language that feels:
- polite
- practical
- easy to understand
- collaborative rather than demanding

Avoid:
- vague requests
- asking too many things in an unstructured wall of text
- language that sounds like the customer is at fault for missing information

## Response format

By default, provide:
1. one recommended version
2. one shorter chat version
3. one more structured version for email or formal follow-up
4. a short note on when to use each version
