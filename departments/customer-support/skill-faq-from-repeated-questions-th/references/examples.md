# Examples - Thai FAQ from Repeated Questions

## Example input
ลูกค้าถามบ่อยว่า ใช้งานได้กับ LINE OA ไหม, ต้องติดตั้งยากไหม, ทดลองใช้ได้หรือเปล่า

## Example output
### FAQ: ระบบนี้ใช้กับ LINE OA ได้ไหม?
ได้ ระบบรองรับการใช้งานร่วมกับ LINE OA โดยสามารถปรับรูปแบบการเชื่อมต่อให้เหมาะกับลักษณะการใช้งานของธุรกิจได้

### FAQ: ต้องติดตั้งยากไหม?
โดยทั่วไปไม่ต้องตั้งค่าซับซ้อนมาก ทีมงานจะช่วยแนะนำขั้นตอนที่จำเป็นให้ตามรูปแบบธุรกิจของคุณ
