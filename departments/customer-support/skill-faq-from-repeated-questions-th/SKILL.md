# Thai FAQ from Repeated Questions

Convert repeated customer questions into reusable FAQ content.

## Output requirements
- Write in Thai
- Make each answer clear and easy for normal users to understand
- Keep answers concise but complete enough to reduce repeat questions
- Use consistent structure across FAQ entries

## Working method
1. Group similar questions together
2. Identify the real confusion behind each question
3. Write a short, direct answer
4. Add important conditions or limits when needed
5. Suggest FAQ titles that are easy to scan

## Default output structure
1. FAQ title
2. common question
3. answer
4. notes / exceptions if needed
