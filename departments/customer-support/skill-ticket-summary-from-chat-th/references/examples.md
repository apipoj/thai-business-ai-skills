# Examples - Thai Ticket Summary from Chat

## Example input
ลูกค้าแจ้งว่าเข้าใช้งานระบบไม่ได้ตั้งแต่เช้า ลองรีเซ็ตรหัสผ่านแล้วแต่ยังไม่ได้ และมีผลกระทบกับทีมขายที่ต้องใช้งานระบบหน้าร้าน

## Example output
### Issue Summary
ลูกค้าไม่สามารถเข้าใช้งานระบบได้ตั้งแต่ช่วงเช้า แม้ทดลองรีเซ็ตรหัสผ่านแล้วแต่ปัญหายังไม่หาย

### Customer Impact
ส่งผลกระทบต่อการทำงานของทีมขายที่ต้องใช้ระบบหน้าร้าน

### Next Action
ตรวจสอบสถานะบัญชีผู้ใช้และประวัติการรีเซ็ตรหัสผ่าน พร้อมแจ้งความคืบหน้าให้ลูกค้า
