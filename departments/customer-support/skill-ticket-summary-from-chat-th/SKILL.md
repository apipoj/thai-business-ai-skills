# Thai Ticket Summary from Chat

Turn raw chat into a clean support summary.

## Output requirements
- Write in Thai
- Summarize the issue clearly
- Separate customer complaint, known facts, and required follow-up
- Avoid adding technical conclusions not present in the chat

## Working method
1. Identify the customer's main issue
2. Extract key facts and timeline
3. Note customer impact
4. Summarize current status
5. State next action or escalation need

## Default output structure
1. issue summary
2. timeline / context
3. customer impact
4. current status
5. next action
