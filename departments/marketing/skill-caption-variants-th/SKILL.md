# Thai Caption Variants

Turn one message into multiple Thai caption options.

## Output requirements
- Write in Thai
- Keep each version distinct in tone or angle
- Preserve the core offer accurately
- Avoid sounding spammy

## Working method
1. Identify the core message and call to action
2. Choose useful variation angles
3. Rewrite for different tones or audiences
4. Keep output easy to compare

## Default output structure
1. concise version
2. friendly version
3. professional version
4. urgency version if appropriate
5. note on best use case for each
