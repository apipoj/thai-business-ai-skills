# Thai Professional Email Reply

Write professional Thai email replies that are clear and easy to send.

## Output requirements
- Write in Thai
- Keep the reply structured and polite
- Match the original context and response goal
- Avoid overlong paragraphs
- End with a clear next step when relevant

## Working method
1. Identify what the sender wants
2. Identify what the reply should achieve
3. Draft a clear response
4. Add any needed clarification or request
5. Close professionally

## Default output structure
1. subject suggestion if useful
2. recommended email reply
3. shorter version if requested
